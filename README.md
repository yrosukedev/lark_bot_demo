# lark_bot_demo
A demo for exploring all the technical details for building a 3rd-part lark application.

## What it does
- Bridging the gap between the Tencent FAAS context and the event dispatcher from Lark Golang SDK
- Registering event listeners to the Lark events through Lark Golang SDK
- Sending Lark events details to a Lark user through the Lark Bot
